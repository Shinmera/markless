Introduction

Block Components

A block is a component that may contain text or other components and always takes up the full width of the document in the output.

A block may span any number of lines, but must always begin and end at the beginning and end of a line respectively.

Glossary

Document

A document is a tree of components that form an abstract representation of the Markless structure.

Component

A unit that specifies a set of representation rules for its contents.

Character

A character is the atomic unit of a string. Characters must be comparable by identity.

Matching

The act of identifying a certain string.

Pattern

Patterns are functions that match regions of a string according to a fixed set of rules.

Syntax

A syntax is a combination of patterns.

String

A strictly ordered sequence of characters. Any sub-sequence of a string is also a string.

Newline

The newline is a specific character that is usually visually interpreted as a line break.

EOL

EOL is any position within a string that is taken by a newline character, as well as the last character in the string.

BOL

BOL is any position within a string immediately after a newline character, as well as the first character in the string.

Markup

A syntax that associates certain sub-sequences in a string with components.
