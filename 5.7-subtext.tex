\definesubsection{Subtext}
\defineidentifier{subtext}{v<start [(]+><content ![(].*><start>}
\definetextualcomponent{subtext}{vertical-align: sub} \\

Subtext is a \g{surrounding inline directive}. It marks the \g{text} to belong to a \g{textual component} that sets the style of the text to appear smaller and below the default text line. Only the \g{text} held by the \g{content binding} is outputted to the \g{resulting textual component}. \\

\begin{examples}
  \example{This is an example v(just so you know)}{This is an example \raisebox{-.4ex}{\scriptsize just so you know}}
\end{examples}
